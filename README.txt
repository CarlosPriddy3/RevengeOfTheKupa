#RevengeOfTheKupa

#Getting Started
-Access the Revenge Of The Kupa exe in the build folder
	-Proceed through menus to start the game (Credits and Controls are non-working
		links
-The objective is to find the cloning machine and shut it down!
-Kupa can shell spin by holding fire3 or shift. This can kill marios but fireballs will still
	hurt
-Kupa can jump over obstacles and over marios and fireballs if timed correctly
-Careful! Mario will get quicker as you progress through the level
